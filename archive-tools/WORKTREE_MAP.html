Open your file explorer and navigate to the tetris-branches folder. Each subfolder contains a complete checkout of that branch.

Drag any HTML file from the worktree folders directly into your browser, or use:

cd tetris-branches/opus-tetris-innovations && for f in *.html; do open "$f"; done
